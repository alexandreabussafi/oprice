🚀 Rotina de Atualização (Terminal VSCode)

Identificamos que o seu terminal não reconhece o comando "git" diretamente. Use os comandos abaixo que já incluem o caminho completo para garantir que tudo funcione.

Preparar Arquivos

Avisa ao sistema para embrulhar todas as alterações feitas.

Copie e Cole & "C:\Program Files\Git\bin\git.exe" add .

Registrar Modificações

Cria uma "etiqueta" com a data e descrição do que foi feito.

Copie e Cole & "C:\Program Files\Git\bin\git.exe" commit -m "feat: atualizacao de componentes e layout"
DICA: Altere o texto entre aspas para descrever sua mudança.

Enviar para Produção

Envia o código para o GitHub e aciona o deploy automático na Vercel.

Copie e Cole & "C:\Program Files\Git\bin\git.exe" push origin main
NOTA: Pode ser solicitado login na primeira vez.
